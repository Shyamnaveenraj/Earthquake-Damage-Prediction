# 🌍 Earthquake-Damage-Prediction

---

## 📊 Overview

This project analyzes data from the **2015 Gorkha earthquake (Nepal)** to understand what causes building damage.
A machine learning model is built to predict damage levels:

* Low
* Medium
* Complete Destruction

---

## 🛠️ Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* SQLite
* Scikit-learn, XGBoost
* Power BI

---

## 🔍 Key Insights

* **Foundation Type:** Type ‘r’ buildings had more damage → need retrofitting
* **Building Age & Size:** Older and bigger buildings were more damaged
* **Location:** Areas near epicenter had highest damage

---

## 🤖 Model

* Used Decision Tree, Random Forest, XGBoost
* Evaluated using Accuracy, Precision, Recall

---

## 🚀 How to Run

1. Clone the repo
2. Open `Earthquake_Analysis.ipynb`
3. Run all cells
4. Open `.pbix` file in Power BI for dashboard
